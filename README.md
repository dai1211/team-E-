# 放課後予定共有アプリ

## プロダクトゴール
友達の放課後の予定が一目でわかり、会うタイミングをすぐ決められるようにする。

## あると嬉しい価値
- 今だれが空いているかがすぐわかる
- 集まる場所や時間を決めやすい
- 予定確認から連絡までを1つのアプリで完結できる

## 主要機能
- チャット
	- グループ内で会話できる
- カレンダー
	- 予定を登録して共有できる
- グループ作成
	- 仲良い友達ごとに予定をまとめられる
- 電話機能
	- すぐ話したいときに連絡できる
- 位置情報共有
	- 近くにいるかどうかを確認できる

## 最初に作るMVP
- グループ作成
- カレンダーで予定共有
- チャット
- 位置情報共有のオン・オフ

## 想定シーン
1. 友達グループを作る
2. それぞれが放課後の予定を入れる
3. 空いている人を見て集合時間を決める
4. チャットで細かい相談をする
5. 必要なら電話で最終確認する
